制作物について

〜Memo Board（React個人開発）〜

⚫︎使用技術: React, TypeScript, dnd-kit, HTML, CSS

 GitHub  実際のアプリ

【課題と解決】
「直感的なアイデア整理」を目的として開発。操作性を追求し、軽量でモダンなライブラリdnd-kitを採用してスムーズなドラッグ＆ドロップを実現しました。また、付箋が追加ボタンや画面外と重ならないよう、独自の衝突判定ロジック（Collision Detection）を組み込み、ストレスのないUXを提供しています。

【技術的アプローチ】
保守性を高めるためにTypeScriptを採用し、型定義（StickyNoteType）による堅牢な設計を行いました。機能面ではuseEffectを用いたウィンドウリサイズ監視による座標の自動補正機能や、ロジックとUIを分離したコンポーネント設計（Container/Presentationalパターン）を取り入れ、拡張性と可読性を意識した実装を行っています。
